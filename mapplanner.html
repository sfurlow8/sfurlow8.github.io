Map Planner

This assignment was completed as a final project in Racket for a class covering data structures and algorithms (CS214). The map planner takes map data with location coordinates and roads and returns responses to two possible queries: find-nearby and find-route. The find-route query returns the shortest path between a given location coordinate and a desired point-of-interest, and the find-nearby query returns the n closest points-of-interest in a given category to a given starting location. Download this zip file to view the algorithm and perform queries.
